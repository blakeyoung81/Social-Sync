# Timeline Keyboard Shortcuts & Features Guide

## Overview
The enhanced VideoClipTimeline component now includes comprehensive keyboard shortcuts for professional video editing workflow, making it fast and efficient to navigate and edit your content.

## Keyboard Shortcuts

### Playback Controls
| Key | Action | Description |
|-----|--------|-------------|
| `Space` | Play/Pause | Toggle video playback |
| `1` | 1x Speed | Set playback speed to normal |
| `2` | 0.5x Speed | Set playback speed to half |
| `3` | 0.25x Speed | Set playback speed to quarter |
| `4` | 1.5x Speed | Set playback speed to 1.5x |
| `5` | 2x Speed | Set playback speed to double |

### Navigation
| Key | Action | Description |
|-----|--------|-------------|
| `←` | Step Back | Move 1 second backward |
| `→` | Step Forward | Move 1 second forward |
| `↑` | Jump Back | Move 10 seconds backward |
| `↓` | Jump Forward | Move 10 seconds forward |
| `Home` | Go to Start | Jump to beginning of timeline |
| `End` | Go to End | Jump to end of timeline |

### Editing
| Key | Action | Description |
|-----|--------|-------------|
| `S` | Split Clip | Split selected clip at current time |
| `Delete` | Delete Clip | Delete the selected clip |
| `Backspace` | Delete Clip | Alternative delete key |
| `Ctrl+Z` | Undo | Undo last action (placeholder) |
| `Ctrl+S` | Save | Save project (placeholder) |

### Zoom & View
| Key | Action | Description |
|-----|--------|-------------|
| `+` | Zoom In | Increase timeline zoom level |
| `-` | Zoom Out | Decrease timeline zoom level |

## Mouse Controls

### Timeline Interaction
- **Click**: Seek to position
- **Drag**: Scrub through timeline
- **Scroll**: Horizontal scroll when zoomed

### Clip Manipulation
- **Click Clip**: Select clip
- **Drag Clip**: Move clip to new position
- **Drag Left Edge**: Trim clip start
- **Drag Right Edge**: Trim clip end
- **Double Click**: Open clip properties (future feature)

### Timeline Navigation
- **Mouse Wheel**: Scroll timeline horizontally
- **Ctrl + Mouse Wheel**: Zoom in/out (future feature)

## Troubleshooting

### Common Issues
1. **Shortcuts not working**: Make sure timeline is focused
2. **Slow performance**: Reduce zoom level or close other apps
3. **Playhead not visible**: Click timeline to focus
4. **Clips not selecting**: Click directly on clip content

### Performance Tips
1. Use lower zoom levels for smoother scrolling
2. Close unnecessary browser tabs
3. Use hardware acceleration if available
4. Optimize video file formats for better performance
